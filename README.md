# Race Into The Shadow

## Project Outline
- 개발 기간: 2026.01.16 ~
- 개발 인원: 1명
- 주요 목표: **빛이 닿으면 사라지는 발판을 피해, 움직이는 그림자 길을 따라 결승점에 도달하라.**
  - 플레이어는 계속해서 움직이는 광원(거대한 태양, 흔들리는 램프 등)에 의해 실시간으로 변하는 **그림자 영역**만을 밟고 이동해야 합니다.
  - 빛이 닿은 발판은 '유령(Ghost)' 상태가 되어 통과해버리고(추락), 그림자가 드리워진 발판만 '고체(Solid)'가 되어 밟을 수 있습니다.

## Game Scenario
먼 미래, 태양의 이상 활동으로 인해 '빛' 자체가 물질을 분해하는 성질을 갖게 된 디스토피아 세계입니다.

인류는 지하로 숨어들었지만, 지상에 남겨진 귀중한 에너지 코어(데이터)를 회수하기 위해 '러너(Runner)' 드로이드를 파견합니다. 거대한 구조물들이 회전하며 만들어내는 일시적인 그림자의 길을 예측하고, 빛의 파도를 피해 안전한 그림자 영역(Safety Zone)을 찾아 끊임없이 달려야 합니다. 당신은 가장 빠른 경로를 계산해 코어를 회수하고 복귀해야 하는 러너입니다.

## Game Flow
### **Phase 1: 잠입 (Infiltration) - "그림자를 밟아라"**
- **분위기:** 정적, 신비로움, 긴장감 (BGM: 조용하고 낮은 템포)
- **규칙:** [그림자 = 고체 / 빛 = 통과(낙사)]
- **플레이:**
    - 광원이 천천히 규칙적으로 움직입니다.
    - 플레이어는 타이밍을 계산하며 '그림자 길'을 찾아 천천히 위로(또는 안으로) 이동합니다.
    - 퍼즐을 풀듯 신중하게 점프해야 합니다.

#### **⚡ 전환점 (The Turning Point)**
- 플레이어가 맵 끝에 있는 '**에너지 코어**'를 획득합니다.
- **이벤트:**
    - 화면이 붉게 점멸하며 경보음이 울립니다.
    - 시스템 음성: "코어 이탈 감지. 보안 프로토콜 반전(Inverted) 가동."
    - 월드의 규칙이 뒤집힙니다.

### Phase 2: 탈출 (Escape) - "빛을 쫓아라"
- **분위기:** 급박함, 혼란, 붕괴 (BGM: 빠르고 강렬한 비트)
- **규칙:** [빛 = 고체 / 그림자 = 통과(낙사)] 로 반전(Inversion) 됩니다.
- **플레이:**
    - 방금까지 안전하게 밟고 왔던 그림자 영역이 이제는 떨어지는 구멍이 됩니다.
    - 플레이어는 방금 왔던 길을 되돌아가야 하는데, 이번엔 피했던 '빛'만 밟아야 하므로 전혀 다른 길처럼 느껴집니다.
    - 난이도를 높이기 위해 광원의 이동 속도가 1.5배 빨라지거나, 맵이 아래서부터 무너지기 시작합니다.

## 게임 루프
1. **시작**: 플레이어는 Start Point에서 스폰됩니다.
2. **잠입 (Phase 1)**: 회전하는 광원을 피해 '**그림자**'만 밟으며 맵 끝까지 이동합니다.
3. **획득**: 맵 끝에 있는 Energy Core와 상호작용합니다.
4. **반전 (Phase 2)**: 경보가 울리며 규칙이 바뀝니다. 이제 '**빛**'만 밟으며 Start Point(탈출구)로 되돌아옵니다.
5. **승리**: 탈출구에 도달하면 게임 클리어.

## 레벨 디자인 (MVP용 테스트 맵 구성)

복잡한 맵 대신, 하나의 직선형 복도를 '**장애물 달리기 코스**'처럼 구성합니다.
- **구역 1 (회전 풍차):**
    - 중앙에 거대한 프로펠러가 돌고 있고, 뒤에서 강한 빛이 비칩니다.
    - 프로펠러 그림자가 시계 방향으로 돕니다. 플레이어는 그림자를 따라 원형 발판을 건너야 합니다.
- **구역 2 (점멸하는 다리):**
    - 주기적으로 깜빡이는 조명 아래 놓인 다리입니다.
    - Phase 1에서는 조명이 꺼졌을 때 건너고, Phase 2에서는 조명이 켜졌을 때 건너야 합니다.
- **구역 3 (코어 안치소):**
    - 가장 안쪽 깊은 곳. 코어를 집는 순간 조명 색이 붉은색(비상 상황)으로 바뀌며 분위기를 반전시킵니다.
